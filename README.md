# تطبيق خدمات راتلوزن (Ratlozen Services)

تطبيق أندرويد متكامل لمتجر إلكتروني يوفر خدمات رقمية متنوعة مثل بطاقات الألعاب والاتصالات والخدمات الأخرى.

## المميزات الرئيسية

- **الصفحة الرئيسية**: عرض المنتجات والفئات المختلفة مع واجهة مستخدم جذابة
- **سلة المشتريات**: إدارة المنتجات المختارة مع عرض رسالة ودية عند كون السلة فارغة
- **الدعم والأسئلة الشائعة**: قسم متخصص للإجابة على الأسئلة الشائعة
- **محفظة المستخدم**: إدارة الرصيد والمحفظة الرقمية
- **الملف الشخصي**: إعدادات المستخدم والشروط والأحكام

## التقنيات المستخدمة

- **Flutter 3.22.2**: إطار العمل الرئيسي لتطوير التطبيق
- **Dart 3.4.3**: لغة البرمجة
- **Material Design 3**: نظام التصميم الحديث

## متطلبات التشغيل

- Flutter SDK 3.22.2 أو أحدث
- Dart SDK 3.4.3 أو أحدث
- Android SDK (لتطوير تطبيقات أندرويد)
- Java Development Kit (JDK)

## التثبيت والتشغيل

### 1. استنساخ المستودع
```bash
git clone https://github.com/Ratluzen2/Ratlozen-Services.git
cd Ratlozen-Services
```

### 2. تثبيت الحزم
```bash
flutter pub get
```

### 3. تشغيل التطبيق
```bash
flutter run
```

### 4. بناء ملف APK
```bash
flutter build apk --release
```

سيتم إنشاء ملف APK في المسار: `build/app/outputs/flutter-apk/app-release.apk`

## الشاشات الرئيسية

### 1. الصفحة الرئيسية (Home)
- عرض شريط البحث والسلة
- عرض العروض الترويجية
- عرض الفئات المختلفة (الجميع، الاتصالات، الألعاب، متاجر التطبيقات)
- عرض المنتجات في شكل شبكة

### 2. سلة المشتريات (Cart)
- عرض المنتجات المختارة
- رسالة ودية عند كون السلة فارغة
- زر للعودة إلى الصفحة الرئيسية

### 3. الدعم (Chat)
- قسم الأسئلة الشائعة
- أسئلة قابلة للتوسع والطي

### 4. الملف الشخصي (Profile)
- عرض الرصيد الحالي
- خيارات إضافة رصيد وشحن البطاقات
- الإعدادات والشروط والأحكام
